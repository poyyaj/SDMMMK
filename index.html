Genome Analysis

  • View genome assembly data for both species.
  • Check genome annotation features for the insulin gene.
  • Identify coding regions and exons.

Protein Exploration

  • Explore insulin protein structure (UniProt).
  • Compare sequence features (signal peptide, chains).
  • Note differences in length or active sites.

Mutation Detection with BLAST

  • Use NCBI BLAST to align human and rat insulin genes.
  • Observe mismatches or gaps (possible SNPs).
  • Highlight biologically significant regions.

Multiple Sequence Alignment

  • Perform MSA using ClustalX or online tools.
  • Align insulin sequences from humans, rats, and mice.
  • Visualize conserved and variable regions.

Phylogenetic Tree Construction

  • Import aligned sequences into MEGA 6.
  • Construct a phylogenetic tree.
  • Interpret evolutionary relationships among species.

Scoring & Algorithms

  • Understand PAM and BLOSUM scoring matrices.
  • Learn basics of dynamic programming for alignment.
  • Discuss global vs. local alignment approaches.

Building a Simple Bioinformatics Database

  • Use MS Access to create a gene info database.
  • Store gene name, species, sequence, and accession number.
  • Design simple queries and forms.

Applications in Real World

  • Health: Diabetes diagnosis using insulin gene variants.
  • Agriculture: Genetic improvement in animals.
  • Environment: Gene markers for stress adaptation.

Wrap-up: The human-rat insulin gene comparison provides hands-on insight into real-world applications of bioinformatics.
